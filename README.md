## How to compile and run

1. Clone this repository
   ```bash
   git clone https://github.com/cypotat/Java-Labs.git
   cd Java-Labs
   ```
2. Compile the specific lab

   ```bash
   cd src
   javac labs/sem1/lab1/*.java
   ```

3. Run it!
   ```bash
   java labs.sem1.lab1.Main
   ```
